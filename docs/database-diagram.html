Core Structure: GRADES ‚Üí SUBJECTS ‚Üí CHAPTERS ‚Üí SECTIONS ‚Üí LESSONS/EXERCISES
User Levels: Tracked per subject, calculated based on exercise history
Assignments: Can be assigned to individual users or entire classes
Submissions: Supports both multiple choice (JSON answers) and essay (image uploads)
Notifications: Supports email, SMS, and push notifications with scheduling
AI Integration: SUBMISSIONS.ai_feedback stores OpenAI API response for essay grading
Authentication: Users login with username (email is optional, used for notifications only)
Conversations: Each essay submission creates a conversation for review/explanation. Users can also have free chat conversations
Messages: All messages in conversations are stored for history and context
Conversation Configs: Configurable response formats and system prompts for different conversation types
